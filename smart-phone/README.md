# متجر المنتجات التقنية

متجر إلكتروني متخصص في عرض وطلب المنتجات التقنية.

## المميزات

- واجهة مستخدم حديثة ومتجاوبة
- تصنيف المنتجات إلى أقسام وعلامات تجارية
- سلة تسوق ديناميكية
- إرسال الطلبات مباشرة إلى المالكين
- دعم متعدد الأجهزة (كمبيوتر، هاتف، لوحي)
- حماية متقدمة للنماذج
